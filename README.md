SentiMap은 원래 NHN NEXT 팀 프로젝트로 만들었던 작품입니다.

##서비스 개요  
간단한 조작만으로 순간의 감정을 표현할 수 있고,  
지도 위에서 전세계인들과 
시각화된 감정을 실시간으로 공유 가능한 위치 기반 SNS
